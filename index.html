Magic the Gathering Preconstructed Products
Download CSV files for preconstructed decks from the following Dropbox links, then go to Inventory > Add Cards > From a CSV File and import the deck. Please note what cards each CSV contains prior to importing, in case you would like to edit the CSV first (e.g. split up a Duel Deck with a friend, add tokens to a Commander deck, remove the basic lands, etc.).

Key:
  • 100% accurate CSV
  • Unverified CSV This usually just means the basic lands may not be split correctly between the different artworks
  • CSV not created yet
